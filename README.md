# Hello World

    This is for testing purpose only

### 📦 Dependency
  - jQuery `3.4.1`
  - Bootstrap `4.4.1 `
  - Animate.css `3.7.2`
  - FontAwesome `5.12.1`
  - Feather Icons `4.26.0`

### 🧰 Requirement
  - Node / NPM
  - Yarn (Optional)

```bash
> npm install -g gulp-cli
```

### 📐 Installation
```bash
> git clone https://github.com/ivan8i/hello-world.git
> cd hello-world
> npm install
# or
> yarn install
```

### 🔍 Usage
```bash
# Start Development
> npm start
# or
> yarn start

# Rebuild
> npm run rebuild
# or
> yarn run rebuild

# Production
> npm run build
# or
> yarn run build
```
